* RT3883/3662 CROSS-TOOLCHAIN BUILD INSTRUCTION *

To build the cross-toolchain, you need Linux environment. Debian squeeze 6.0.3 and 
Ubuntu 10.04 distros has been tested.

Just run build script "build_toolchain" and wait for the build process.


* NOTE *

To build the cross-toolchain under Debian squeeze you need the packages:
- build-essential
- gawk
- sudo
- pkg-config
- gettext
- automake
- autoconf
- bison
- flex
- libgmp3-dev
- libmpfr-dev
- libmpc-dev
